- [`task1`](https://github.com/CostaVaryn/myDraft/tree/master/src/test/java/stepik/interactiveSQLtrainer/SQLqueries/block5/task1.sql) - включение нового клиента в базу данных;

- `task8` - фиксация дат прохождения каждого этапа заказа (начало этапа);
- `task9` - фиксация дат прохождения каждого этапа заказа (завершение этапа).
